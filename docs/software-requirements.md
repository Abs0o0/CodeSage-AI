# CodeSage - Software Requirements Specification (SRS)

## 📌 Project Summary
CodeSage is a web-based platform designed to analyze, review, and enhance frontend code using AI-powered tools.

## 🏗️ System Overview
- **Frontend:** React (Vite) + Tailwind + Material UI  
- **Backend:** Node.js + Express  
- **Database:** MongoDB  
- **Authentication:** JWT + Refresh Tokens  
- **AI Integration:** External APIs  

## 🚨 Problem Statement
Many developers struggle with writing clean, maintainable, and well-structured code. CodeSage provides AI assistance to improve code quality.

## 🎯 Objectives
- Provide automated code review  
- Generate documentation  
- Improve code quality  
- Support collaboration  

## 👥 Stakeholders
- Developers  
- Admins  
- Reviewers  
- Project Team  

## ⚙️ Functional Requirements
- User authentication (login/register)
- Project management
- Code upload functionality
- AI-based code analysis
- Code rewriting suggestions
- Collaboration features
- Export results

## 📊 Non-Functional Requirements
- Performance
- Scalability
- Usability
- Maintainability
- Security

## 🔄 Use Cases
- Upload Code
- Request Review
- View Results
- Collaborate with team members

## 🏛️ System Architecture
- Client-Server architecture  
- React frontend  
- Node.js backend  

## 🤖 AI Review Flow
1. User submits code  
2. Code is queued  
3. AI processes the code  
4. Results are displayed  

## ⚠️ Constraints
- Maximum file upload size: 1MB  
- Dependency on external AI APIs
